==Bash uses keyboard shortcuts to help you navigate and edit commands faster==. Most of these are based on **Emacs** keybindings, which is the default for most Linux and macOS terminals.

Moving the Cursor

- **`Ctrl + A`**: Jump to the **beginning** of the line.
- **`Ctrl + E`**: Jump to the **end** of the line.
- **`Alt + B`**: Move backward **one word**.
- **`Alt + F`**: Move forward **one word**.
- **`Ctrl + XX`**: Toggle between the **start** of the line and your **current** cursor position. 
- 
Editing & Deleting

- **`Ctrl + U`**: Cut/delete from the cursor back to the **start** of the line.
- **`Ctrl + K`**: Cut/delete from the cursor to the **end** of the line.
- **`Ctrl + W`**: Cut/delete the **word** before the cursor.
- **`Alt + D`**: Delete the **word** after the cursor.
- **`Ctrl + Y`**: **Paste** (yank) the last text you cut.
- **`Ctrl + T`**: Swap (transpose) the last two characters. 

Command History

- **`Ctrl + R`**: Search your command **history** (reverse-i-search).
- **`Ctrl + G`**: Exit history search mode without running a command.
- **`Ctrl + P` / `Ctrl + N`**: Go to the **previous** or **next** command in history (same as up/down arrows).
- **`!!`**: Run the **previous command** again.
- **`Alt + .`**: Insert the **last argument** from your previous command. 
Process Control

- **`Ctrl + C`**: **Kill** or interrupt the currently running process.
- **`Ctrl + Z`**: **Suspend** (pause) the current process and put it in the background.
- **`Ctrl + L`**: **Clear** the screen while keeping your current line.
- **`Ctrl + D`**: **Exit** the current shell or send an EOF (End-of-File) marker.
